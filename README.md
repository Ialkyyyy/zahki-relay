# zahki-relay

Expose localhost to the internet. Inspect incoming requests. Replay them for debugging.

https://zahkirelay.alkhabaz.dev

## Features

- Localhost tunneling via WebSockets
- Request inspector (headers, body, params)
- One-click replay
- CLI tool

## Run it

```
git clone https://github.com/Ialkyyyy/zahki-relay.git
cd zahki-relay
npm install
npm run dev
```

## Stack

React, Express, WebSockets
